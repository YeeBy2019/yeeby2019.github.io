Markdown语法总结

引言

今天花了两个小时学了一下Markdown的语法（鸽了很久的Markdown，鸽了很久的博客，鸽了很久的github WP

本来是打算昨天就学完的 最后因为配置HexoEditor花了一天的时间，最后因为不熟悉命令行+HexoEditor的确存在BUG，最后选择使用Typora进行博客文章的编辑，然后就有了今天这篇处女博文。

1.Markdown标题

在Markdown中，有两种设置标题的方法：

一种是：

  • Title
  • ============

或

  • Title
  • -——————–

另一种是：

# 一级标题

## 二级标题

### 三级标题

··········

以此类推

我通常使用第二种，因为第一种在Typora里很难打出来（误 ，Typora里直接编辑的时候按一次回车在源代码里默认是两次换行，按Shift+Enter才是一次换行。

2.Markdown段落格式

Markdown的段落没有特殊格式，在一次段落的结尾添加两个空格之后一次换行 或 在段落的下面添加一个空行

即可用作段落与段落之间的换行。

3.Markdown字体

*斜体文本*

**粗体文本**

***粗斜体文本***

注：* 可以用 _ 来替代，效果相同

4.Markdown分隔线

-——————–

**\*****

实测符号连续至少三个就会产生分隔线，这两种分隔线有粗细之分

据说在符号与符号之间加上空格也可以产生分隔线的效果，我没试出来/(ㄒoㄒ)/

5.Markdown删除线

~我身上的就是删除线\

效果：我身上的就是删除线

6.Markdown下划线

Markdown中的下划线是通过HTML中的标签<u>来实现的，实例如下：

<u>\划重点！！！</u>

效果：划重点！！！

7.Markdown脚注

脚注是对文本的补充说明。

嗯，其实就是我们在看书时书页最下方的①②③的东西，在字典中尤为常见。那么在Markdown中要如何实现呢？

格式如下：

[^脚注名称]

[^脚注名称]：脚注要补充的内容

这道题我们的直觉就是用DFS^①搜索一遍

8.Markdown列表

Markdown列表分为有序列表和无序列表，字面意思。

有序列表：

1.第一项

2.第二项

3.第三项

效果：

  1. 第一项
  2. 第二项
  3. 第三项

嗯，你没看错，没变化。

无序列表：

* 第一项

* 第二项

* 第三项

效果：

  • 第一项
  • 第二项
  • 第三项

注：* 可以用 + - 来替代，效果相同

列表嵌套:

有序里面套无序：

1.第一项

​ * 第一项

​ * 第二项

2.第二项

​ * 第一项

​ * 第二项

效果：

1.第一项

* 第一项

* 第二项

2.第二项

* 第一项
  • 第二项

无序里面套有序：

* 第一项

​ 1.第一项

​ 2.第二项

* 第二项

​ 1.第一项

​ 2.第二项

效果：

  • 第一项

    ​ 1. 第一项

    ​ 2. 第二项

  • 第二项

    ​ 1. 第一项

    ​ 2. 第二项

10.Markdown代码

作为一名计算机专业人才（误，当我们使用Markdown写文章的时候，免不了需要大篇幅的展示代码。接下来，我们介绍一下如何使用Markdown来展示我们的代码。

1.如果是小部分的代码区块，例如printf()这个函数,我们可以用如下格式：

` printf()`

效果：printf()

2.如果是一个很大的代码区块，我们可以用如下格式：

​ 代码

什么？你说你没看懂？

其实这里使用了一个制表符(Tab)——4个空格，在编写Markdown源代码时，在你接下来要写的代码区块前打一个Tab，就可以愉快的写代码啦。

3.我们也可以用 ``` 包裹一段代码，并指定一种语言（也可以不指定）：

例如：

```c++

#include <bits/stdc++.h>

using namespace std;

int main()

{

​ int n;

​ cin>>n;

​ cout<<n;

​ return 0;

}

```

11.Markdown链接

格式：

[连接名称](链接地址)

或者

<链接地址>

例如：

百度

或

<www.baidu.com>

注：链接可以用变量来代替，在文档结尾注明，例如：

谷歌

12.Markdown图片

![alt 属性文本](图片地址)

![alt 属性文本](图片地址 “可选标题”)

例如：

![alt W3Cschool](http://www.w3school.com.cn/i/w3school_logo_white.gif\)

注：图片的引用还可以使用HTML中的

<img src=”” width= height= >标签来设置图片的大小。

例：

<img src=”http://www.w3school.com.cn/i/w3school_logo_white.gif" width=100 height=40>
